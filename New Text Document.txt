# Important Commands

```text
RMI Registry Start Command:
rmiregistry
```

```text
Run RMI Server:
java com.rmi.AddServer
```

```text
Run RMI Client:
java com.rmi.AddClient
```

```text
CORBA Naming Service Start:
tnameserv -ORBInitialPort 1050
```

```text
Run CORBA Server:
java ReverseServer -ORBInitialPort 1050
```

```text
Run CORBA Client:
java ReverseClient -ORBInitialPort 1050
```

```text
Run MPI Program:
mpjrun.bat -cp E:\DS\MPILab4\bin -np 4 ArrSum
```

```text
Generate CORBA Java Files from IDL:
idlj -fall ReverseModule.idl
```

```text
Check Java Version:
java -version
```

```text
Check Java Compiler Version:
javac -version
```
